Device Info:

device.cordova:

Cordova version

device.model:

Returns the name of the device's model or product:

     Android:    
     Nexus One -> "Passion" (code name)
     Motorola Droid -> "voles"

     BlackBerry: 
     Torch 9800 -> "9800"

     Browser:    
     Google Chrome -> "Chrome"
     Safari -> "Safari"

     iOS:
     iPad Mini -> iPad2,5
     iPhone 5 -> iPhone 5,1

     Desktop:
     OSX -> "x86_64"
device.platform:

Get the device's operating system name

   - "Android"
   - "BlackBerry 10"
   - "browser"
   - "iOS"
   - "WinCE"
   - "Tizen"
   - "Mac OS X"
device.uuid:

Get the device's Universally Unique Identifier

device.version:

Get the operating system version

device.manufacturer:

Get the device's manufacturer

device.isVirtual:

Whether the device is running on a simulator

device.serial:

Get the device hardware serial number
